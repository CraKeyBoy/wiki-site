# Error 404 Not Found - 页面不见了哟~

---------

![诶？！竟然被你发现了404？！](/images/404.jpg)

您所访问的地址无法找到或者已经被删除了，

如果您是从搜索结果点击进来看到的 404，可以先尝试使用页面 **右上方的搜索文章**。

访问本站出现 404，一般是因为以下原因：

本站启用固定链接之后，搜索引擎未全面更新收录。所以有时候搜索结果里的文章链接是错误的。

**来都来了，就先逛逛嘛~~**

您可以访问 [**本站主页**](/) ，或者联系我，吐槽下问题的详细情况。
